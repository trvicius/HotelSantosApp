## 🚀 Instalando

Para instalar o Projeto, siga estas etapas:

Linux e macOS:
```
1. Clone o repositorio
2. <expo install>
```

Windows:
```
1. Clone o repositorio
2. <expo install>
```

## ☕ startando o projeto

Para startar o sistema HotelSantos, siga estas etapas:

```
expo start
```
